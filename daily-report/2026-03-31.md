Highlights include a new `CLAUDE_CODE_NO_FLICKER=1` environment variable for flicker-free rendering, a `PermissionDenied` hook that fires when auto mode classifier denials occur, a fix for prompt cache misses in long sessions, a fix for nested CLAUDE.md files being re-injected multiple times, a fix for Edit/Write tools doubling CRLF line endings on Windows, and a critical fix for a `StructuredOutput` schema cache bug that was causing ~50% failure rates.
