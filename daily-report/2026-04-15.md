# Claude Code Daily Report — April 15, 2026

> Trending articles, discussions, and resources about Claude Code from the previous day.

## Highlights

April 14 was a major launch day for Claude Code: Anthropic officially shipped two features that had been anticipated since the "Epitaxy" leak — cloud-based **Routines** (scheduled and event-triggered automations that run on Anthropic's infrastructure without requiring your laptop) and a heavily redesigned **desktop app** with an integrated terminal, drag-and-drop pane layout, and a sidebar for managing parallel sessions. Coverage from SiliconANGLE, 9to5Mac, The Register, and The New Stack confirmed the broad industry attention, while a 584-point Hacker News thread surfaced significant developer anxiety about vendor lock-in and platform trust.

## Official Announcements

**[Introducing routines in Claude Code](https://claude.com/blog/introducing-routines-in-claude-code)** — Anthropic launched Routines as a research preview: saved Claude Code configurations (a prompt, one or more repos, and a set of connectors) that execute on Anthropic's cloud infrastructure on a schedule, via a dedicated API endpoint, or in response to GitHub webhook events. Because they run server-side, your machine doesn't need to stay online. Usage counts against daily limits — Pro: 5/day, Max: 15/day, Team/Enterprise: 25/day — with metered overage billing available for heavier use.

**[Redesigning Claude Code on desktop for parallel agents](https://claude.com/blog/claude-code-desktop-redesign)** — The desktop app received its most significant redesign since launch, built explicitly around the orchestrator model of development. New additions include a session sidebar for filtering and switching between concurrent sessions, an integrated terminal, an in-app file editor, a rebuilt diff viewer optimized for large changesets, an HTML/PDF preview pane, and fully drag-and-drop pane layouts. A new side-chat shortcut (`⌘+;` / `Ctrl+;`) lets you ask questions without interrupting a running agent. SSH support is now extended to Mac (previously Linux-only).
