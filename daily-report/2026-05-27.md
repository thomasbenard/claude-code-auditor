## Tools & Releases

**[Claude Code v2.1.152](https://github.com/anthropics/claude-code/releases)** — Major release (shipped May 27 01:30 UTC) with several developer-facing improvements:

- **`/code-review --fix`** applies review findings directly to your working tree after the review
- **`/simplify`** now invokes `/code-review --fix` under the hood
- **`disallowed-tools` frontmatter** — skills and slash commands can now block specific tools from being available during their execution
- **`/reload-skills`** command re-scans skill directories without restarting the session
- **`SessionStart` hook** can now return `reloadSkills: true` and set the session title via `hookSpecificOutput.sessionTitle`
- **`MessageDisplay` hook** — new event type to transform or suppress assistant message text before display
- **Auto mode** no longer requires opt-in consent
- Vim mode history search, thinking summaries, workflow display improvements
- Bug fixes for terminal styling, plugin handling, and permission prompts
