## Official Announcements

**[Claude Code v2.1.160](https://github.com/anthropics/claude-code/releases)** — Released June 2. Security-focused patch: Claude now prompts before writing to shell startup files (`.zshenv`, `.zlogin`, `.bash_login`, `~/.config/git/`) to prevent unintended command execution. `acceptEdits` mode also prompts before writing build-tool configs that grant code execution (`.npmrc`, `.yarnrc*`, `bunfig.toml`, `.bazelrc`, `.pre-commit-config.yaml`, `.devcontainer/`). Also fixes: single-file grep now satisfies the read-before-edit check, Windows/WSL clipboard via PowerShell interop, background session resumption losing chat history, directory deletion on Windows after `claude rm`, and CJK IME composition. The dynamic-workflow trigger keyword is renamed from `workflow` to `ultracode`.
