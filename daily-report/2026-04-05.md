## Official Announcements

**[Anthropic Says Claude Code Subscribers Will Need to Pay Extra for OpenClaw Usage](https://techcrunch.com/2026/04/04/anthropic-says-claude-code-subscribers-will-need-to-pay-extra-for-openclaw-support/)** — Effective April 4, Anthropic ended subscription-based access to Claude Pro and Max within third-party agentic harnesses like OpenClaw, citing "outsized strain" on infrastructure. Approximately 60% of active OpenClaw sessions ran on subscription credits; those users must now switch to API billing or a new pay-as-you-go add-on. Claude Code (the official CLI) is not affected.
