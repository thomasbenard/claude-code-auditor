**[Starting June 15, claude -p usage will change](https://news.ycombinator.com/item?id=48130374)** — Anthropic's billing restructure takes effect today. Paid plans now include a monthly Agent SDK credit pool (Pro: $20, Max 5x: $100, Max 20x: $200) metered at full API rates. Interactive terminal sessions are unaffected; only programmatic usage (`claude -p`, Agent SDK, GitHub Actions, third-party apps) draws from the new pool. No rollover, no automatic overflow fallback.
