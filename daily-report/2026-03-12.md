## Official Announcements

**[Auto Mode Research Preview](https://x.com/rohanpaul_ai/status/2030156251821392096)** — Anthropic announced Auto Mode for Claude Code (`claude --enable-auto-mode`), expected to begin rolling out March 12. The feature hands permission decisions to Claude itself — for each action, the agent reasons about scope and risk before deciding whether to auto-approve or surface a prompt. Read-only operations on project files are generally auto-approved; shell commands with broad filesystem access or external network calls are more likely to require approval. Anthropic recommends running Auto Mode in isolated environments only.
