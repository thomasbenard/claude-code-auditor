## Official Announcements

**[Claude Code v2.1.139 Release Notes](https://github.com/anthropics/claude-code/releases)** — Anthropic shipped v2.1.139 on May 11 with two headline features: **Agent View** (`claude agents` opens a single list of all sessions — running, blocked, or done) and the **`/goal` command** (set a completion condition; Claude works across turns until it's met, with a live overlay showing elapsed time, turn count, and token usage). Additional additions include `/scroll-speed` for scroll tuning, `continueOnBlock` hook config for `PostToolUse` events (feeds rejection reason back to Claude instead of stopping), `CLAUDE_PROJECT_DIR` env var injected into MCP stdio servers, and a `--plugin-url` flag for fetching plugin `.zip` archives from URLs. VS Code gains **Cmd/Ctrl+Shift+T** to reopen the last closed session tab. Notable bug fixes: expired-credentials deadlock blocking `claude auth`, unbounded memory growth in HTTP/SSE MCP servers, and `Skill(name *)` wildcard permission rules.
