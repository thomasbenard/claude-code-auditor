**[Claude Code 2.1.169 Release](https://releasebot.io/updates/anthropic/claude-code)** — Released June 9, this version adds three notable features: `--safe-mode` (disables CLAUDE.md and plugins for troubleshooting), `/cd` (moves a session to a new working directory without breaking prompt cache), and a `disableBundledSkills` setting to hide bundled skills and built-in slash commands from the model. It also fixes CPU usage during streaming, enterprise MCP policy enforcement on reconnect, macOS UI stalls, and Windows slash-command scanning performance.
