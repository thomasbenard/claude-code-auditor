## Official Announcements

**[Claude Code v2.1.76](https://code.claude.com/docs/en/changelog)** — Released March 14. Key additions: MCP elicitation support (MCP servers can request structured mid-task input via form dialog or browser URL), new `Elicitation` and `ElicitationResult` hooks, `-n`/`--name` CLI flag for naming sessions at startup, `worktree.sparsePaths` setting for sparse-checkout in large monorepos, a `PostCompact` hook that fires after context compaction, and a new `/effort` slash command to tune model effort level. Enterprise admins also get a configurable session quality survey via `feedbackSurveyRate`. Notable fixes include deferred tools losing their schemas after compaction, plan mode asking for re-approval unnecessarily, and a circuit breaker for runaway auto-compaction retries.

**[Claude March 2026 Usage Promotion](https://support.claude.com/en/articles/14063676-claude-march-2026-usage-promotion)** — Running March 13–27, Anthropic is doubling usage limits for all Free, Pro, Max, and Team plan users during off-peak hours (outside 8 AM–2 PM ET / 5–11 AM PT). The bonus applies across Claude, Claude Code, Cowork, and the Office integrations. Enterprise plans are excluded. No action needed — the bonus activates automatically.
