**[Anthropic Ships Claude Code Channels — Telegram and Discord Integration](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)** — Developers can now connect Claude Code to a personal Telegram bot or Discord server. Once linked via `/telegram:configure`, the phone becomes a remote control for the dev environment: send instructions from anywhere, receive output asynchronously. The `--channels` permission relay (added in v2.1.81) forwards tool-approval prompts to the mobile device so unattended runs don't stall waiting for permission.
