Key additions: a `managed-settings.d/` drop-in directory alongside `managed-settings.json` for composable policy fragments across teams; `CwdChanged` and `FileChanged` hook events enabling reactive environment tooling (e.g., auto-loading direnv when the working directory changes); transcript search; pasted-image position chips (`[Image #N]` inserted at the cursor for positional references in prompts); and a `sandbox.failIfUnavailable` setting that exits with an error rather than silently running unsandboxed.
