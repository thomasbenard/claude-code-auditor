# Claude Code Daily Report — June 13, 2026

> Trending articles, discussions, and resources about Claude Code from the previous day.

## Highlights

The dominant story heading into the weekend is the **June 15 subscription split**: starting Sunday, programmatic usage (`claude -p`, Agent SDK, GitHub Actions) draws from a separate monthly credit pool — and the Hacker News community is not happy about it. On the release front, v2.1.173 shipped June 11 with a Fable 5 naming fix and a Windows warning cleanup, following v2.1.172's major nested sub-agent feature drop on June 10.

## Official Announcements

**[Anthropic Splits Claude Subscriptions: What Changes for Indie Hackers on June 15](https://devtoolpicks.com/blog/anthropic-splits-claude-subscriptions-agent-sdk-credit-june-2026)** — Starting June 15, `claude -p`, Agent SDK, GitHub Actions, and third-party agents move to a separate per-user monthly credit pool (Pro gets $20/mo, Max 5x gets $100/mo, Max 20x gets $200/mo). Credits are non-rollover and billed at API rates. Interactive Claude Code sessions in terminal/IDE are unaffected. Anthropic cited unsustainable cost dynamics from subscribers bypassing prompt caching via automated agents.
