## Official Announcements

**[Claude Code v2.1.126](https://github.com/anthropics/claude-code/releases)** — Released May 1 at 02:05 UTC by @ashwin-ant. Key additions:
- **`claude project purge [path]`** — deletes all Claude Code state for a project; supports `--dry-run`, `-y`, `-i`, and `--all` flags.
- **Gateway model picker** — `/model` now lists models from your gateway's `/v1/models` endpoint when `ANTHROPIC_BASE_URL` points to an Anthropic-compatible gateway.
- **OAuth paste in terminal** — `claude auth login` now accepts OAuth codes pasted directly into the terminal, fixing auth flows in WSL2, SSH, and container environments.
- **`--dangerously-skip-permissions`** now bypasses prompts for writes to `.claude/`, `.git/`, `.vscode/`, and shell config files.
- **Auto mode spinner** turns red when permission checks stall.
- **Windows PowerShell 7** detection and improved shell handling.
- **Image paste** auto-downscales images larger than 2000px.
- Fixed `allowManagedDomainsOnly` / `allowManagedReadPathsOnly` being silently ignored.
- Improved Agent SDK stability.

## Community Discussions

**[Uber Torches 2026 AI Budget on Claude Code in Four Months](https://news.ycombinator.com/item?id=47976415)** (Hacker News, 378 upvotes) — A deep discussion on token-spending economics and organizational incentives. Top threads identified three budget-burn patterns: long-lived conversations without context compression, parallel subagent fan-outs, and extreme multi-worktree multitasking. A recurring theme: organizations rewarding high token usage as a productivity proxy (Goodhart's Law), and the concern that heavily automated workflows introduce "slop" bugs across large codebases. Notable data point: cached tokens still scale near-linearly because they are charged per request.
