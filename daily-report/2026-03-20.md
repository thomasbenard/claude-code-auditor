## Tools & Releases

**[Cook](https://github.com/rjcorwin/cook)** (270 stars) — The CLI behind the HN front-page post. Installable via `npm install -g @let-it-cook/cli`. Projects configure via `COOK.md` and `.cook/config.json`. Notable: automatic rate-limit recovery that waits and retries instead of bailing when Claude Code's quota resets. Supports Claude Code, Codex, and OpenCode as backends.

**[cursor-claude-personas](https://github.com/ratnesh-maurya/cursor-claude-personas)** — 38 role-based persona packs for Claude Code, Cursor, and VS Code. Each persona is a folder with rules, skills, and agent instructions for a specific domain. Last updated March 18, 2026. MIT-licensed.
