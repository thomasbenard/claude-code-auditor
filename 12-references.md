## Blogs and Newsletters

Written guides, tips, and workflows from experienced Claude Code users.

### ClaudeLog

The leading community resource for Claude Code documentation, experiments, and best practices. Maintained by InventorBlack, a Claude Developer Ambassador. Features a searchable knowledge base and optimization techniques tested in real development scenarios.

- **Link**: [claudelog.com](https://claudelog.com/)

### Shrivu Shankar — "How I Use Every Claude Code Feature"

A comprehensive brain dump covering the entire Claude Code ecosystem: CLAUDE.md files, custom slash commands, subagents, hooks, and GitHub Actions. One of the most thorough single-article references available.

- **Link**: [blog.sshh.io](https://blog.sshh.io/p/how-i-use-every-claude-code-feature)

### Boris Tane — "How I Use Claude Code"

A workflow developed over 9 months of daily Claude Code usage. Emphasizes a research-plan-implement cycle — never letting Claude write code until you've approved a written plan.

- **Link**: [boristane.com](https://boristane.com/blog/how-i-use-claude-code/)

### Builder.io

Multiple Claude Code guides from beginner to advanced: practical tips, CLAUDE.md authoring, a product-manager perspective, and general best practices.

- **Links**: [How I use Claude Code (+ my best tips)](https://www.builder.io/blog/claude-code) · [How to Write a Good CLAUDE.md File](https://www.builder.io/blog/claude-md-guide) · [Beginner Guide](https://www.builder.io/blog/how-to-use-claude-code)

### Sankalp — "A Guide to Claude Code 2.0"

Practical advice on getting better at using coding agents, written from firsthand experience with Claude Code 2.0.

- **Link**: [sankalp.bearblog.dev](https://sankalp.bearblog.dev/my-experience-with-claude-code-20-and-how-to-get-better-at-using-coding-agents/)

### Peter Yang — Creator Economy

AI tutorials and interviews for busy people, read by 140K+ subscribers. Includes written companions to his YouTube tutorials.

- **Link**: [creatoreconomy.so](https://creatoreconomy.so/)

### Harper Reed — "My LLM Codegen Workflow"

Former CTO of Obama's 2012 campaign, now CEO of an AI research firm. Documents a spec-driven TDD workflow: GPT-4o for ideation → reasoning model for specs → Claude Code for implementation. Key insight: "robots love TDD" as the primary counter to hallucination.

- **Link**: [harper.blog](https://harper.blog/2025/05/08/basic-claude-code/)

### Ran Isenberg — "Claude Code Best Practices: Lessons From Real Projects"

AWS Serverless Hero and Principal Cloud Architect at Palo Alto Networks. Grounded in three completed production projects. Counter-narrative to typical AI hype: "your domain expertise is the bottleneck, not the tool." Covers spec-driven development, security gaps AI won't flag, and BMAD integration.

- **Link**: [ranthebuilder.cloud](https://ranthebuilder.cloud/blog/claude-code-best-practices-lessons-from-real-projects/)

- **Anthropic blog**: Subscribe to [claude.com/blog](https://claude.com/blog) for feature announcements
